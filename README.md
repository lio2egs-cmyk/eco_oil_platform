# אקו-סיסטם — מערכת ניהול חכמה לאקו אויל ואקו דיפו

## תיאור הפרויקט

מערכת ניהול תפעולית חכמה לחברת **אקו-אויל ווירומטל בע"מ** — מתקן לטיפול בפסולת תעשייתית מסוכנת בחיפה.
המערכת מחליפה עבודה ידנית עם קבצי Excel נפרדים לכל לקוח, ומרכזת את ניהול שני ענפי החברה במקום אחד.

---

## רקע עסקי

חברת אקו-אויל פועלת בתחום הטיפול בשפכים תעשייתיים מסוכנים. החברה מורכבת משני ענפים:

- **אקו דיפו** — ניהול מחזור חיים של מיכלי ISO ומכליות (קבלה, שטיפה, תיקון, שחרור)
- **אקו אויל** — טיפול ורגולציה של פריקות חומ"ס (חומרים מסוכנים לסביבה)

---

## בעיות שהמערכת פותרת

- עבודה עם עשרות קבצי Excel נפרדים — טבלה אחת לכל לקוח
- הקלדה ידנית של מספרי זיהוי על ידי עובדים עם מחסום שפה
- תיוק ידני של תמונות שהתקבלו בוואטסאפ
- יצירה ידנית של תעודות פריקה ואישורי שטיפה
- חוסר נראות תפעולית ועסקית בזמן אמת

## מודול אקו דיפו

### מה המודול עושה
ניהול מחזור החיים המלא של מיכלי ISO ומכליות:
קבלה ← שטיפה ← בדיקה ← תיקון ← שחרור

### נקודות קצה עיקריות
- `POST /depot/assets` — יצירת נכס חדש
- `POST /depot/wash-cycles` — פתיחת מחזור שטיפה
- `PATCH /depot/wash-cycles/<id>/complete` — סיום שטיפה + תעודה אוטומטית
- `POST /depot/repair-events` — תיעוד תיקון
- `POST /depot/release-documents` — הפקת מסמך שחרור

---

## מודול אקו אויל

### מה המודול עושה
ניהול פריקות חומרים מסוכנים ורגולציה מול משרד איכות הסביבה:
הצהרת יצרן ← אירוע פריקה ← אישור פריקה אוטומטי (PDF)

### נקודות קצה עיקריות
- `POST /eco-oil/disposal-events` — יצירת אירוע פריקה
- `PATCH /eco-oil/disposal-events/<id>/close` — סגירת אירוע + הפקת אישור PDF
- `GET /eco-oil/disposal-events` — רשימת אירועים
- `POST /eco-oil/producer-declarations` — הצהרת יצרן

### זרמי חומרים נתמכים
אמולסיה, בסיס, חומצה, מינרלי (בור מפריד), מינרלי (קוביה), מזוט, מי שטיפה, סניטרי, סניטרי שלוח, צמחי, רכז שפכים

## שכבת Intelligence — סוכני AI

מערכת multi-agent המנתחת נתונים תפעוליים היסטוריים:

**Crew 1 — Data Analyst**
- סוכן ניקוי ואימות נתונים
- סוכן ניתוח דפוסים עונתיים
- סוכן הפקת דוח EDA

**Crew 2 — Data Scientist**
- סוכן זיהוי חומרים בעייתיים
- סוכן ניתוח עלויות ופערי תמחור
- סוכן הפקת המלצות ניהוליות
